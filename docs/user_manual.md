## 安装与运行

### 环境要求

- Java JDK 8 或更高版本
- Ant 构建工具（可选）

### 安装步骤

1. 从 GitHub 克隆仓库：
   ```bash
   git clone https://github.com/yourusername/JavaGameCollection.git
   ```

2. 进入项目目录：
   ```bash
   cd JavaGameCollection
   ```

### 运行游戏

#### 使用命令行运行

1. 编译项目：
   ```bash
   javac -d bin src/**/*.java
   ```

2. 运行特定游戏（以扫雷为例）：
   ```bash
   java -cp bin games.minesweeper.MinesweeperGame
   ```

#### 使用构建工具（Ant）

1. 编译项目：
   ```bash
   ant compile
   ```

2. 运行特定游戏：
   ```bash
   ant run-minesweeper
   ```

## 游戏玩法

### 1. 扫雷游戏

#### 游戏目标
找出所有非地雷格子，避免触碰到地雷。

#### 控制方式
- **左键点击**：翻开格子
- **右键点击**：标记/取消标记地雷
- **双击已翻开的数字**：快速翻开周围所有非地雷格子（当标记的地雷数量等于数字时）

#### 游戏规则
- 数字表示周围8个格子中的地雷数量
- 如果翻开的是地雷，游戏结束
- 成功找出所有非地雷格子即获胜

#### 难度设置
游戏支持自定义难度：
- **初级**：9x9 格子，10 个地雷
- **中级**：16x16 格子，40 个地雷
- **高级**：30x16 格子，99 个地雷
- **自定义**：可自由设置格子大小和地雷数量

### 2. 贪吃蛇游戏

#### 游戏目标
控制蛇吃到尽可能多的食物，使蛇变得更长，获得更高的分数。

#### 控制方式
- **方向键**：控制蛇的移动方向（上、下、左、右）
- **空格**：暂停/继续游戏
- **R键**：重新开始游戏

#### 游戏规则
- 蛇吃到食物后会变长，分数增加
- 撞到墙壁或自己的身体，游戏结束
- 随着蛇变长，移动速度会逐渐增加

#### 分数系统
- 每吃到一个食物，分数增加 10 分
- 连续吃到食物会有额外奖励分数

### 3. 俄罗斯方块游戏

#### 游戏目标
控制下落的方块，使其填满一行或多行，从而消除这些行，获得分数。

#### 控制方式
- **左右方向键**：移动方块
- **下方向键**：加速下落
- **上方向键**：旋转方块
- **空格**：直接落到底部
- **P键**：暂停/继续游戏

#### 游戏规则
- 方块从顶部随机生成并下落
- 当方块无法继续下落后，会固定在当前位置
- 当一行被方块填满时，该行会被消除
- 消除多行会有额外奖励
- 当方块堆叠到顶部时，游戏结束

#### 分数系统
- 消除一行：100 分
- 消除两行：300 分
- 消除三行：500 分
- 消除四行（Tetris）：800 分

### 4. 吃豆人游戏

#### 游戏目标
控制吃豆人吃掉迷宫中的所有豆子，躲避幽灵，完成所有关卡。

#### 控制方式
- **方向键**：控制吃豆人的移动方向
- **空格**：暂停/继续游戏

#### 游戏规则
- 吃到普通豆子：分数增加
- 吃到特殊豆子：暂时可以吃掉幽灵
- 吃到水果：获得额外分数
- 被幽灵碰到：失去一条生命
- 生命值为0时，游戏结束

#### 关卡系统
- 游戏包含多个关卡，每个关卡的迷宫布局和难度不同
- 随着关卡的提升，幽灵的移动速度会增加

### 5. 打砖块游戏

#### 游戏目标
控制挡板反弹球，击碎所有砖块，通关游戏。

#### 控制方式
- **左右方向键**：移动挡板
- **空格**：发射球
- **P键**：暂停/继续游戏

#### 游戏规则
- 球碰到砖块会击碎砖块并反弹
- 球碰到挡板会反弹
- 球掉落底部，失去一条生命
- 生命值为0时，游戏结束
- 击碎所有砖块即可通关

#### 特殊砖块
- **普通砖块**：一击即可击碎
- **强化砖块**：需要多次击打
- **道具砖块**：击碎后会掉落道具，如增加挡板大小、多球等

### 6. 四子连珠游戏

#### 游戏目标
先将4个相同颜色的棋子连成一条线（横、竖、对角线）的玩家获胜。

#### 控制方式
- **鼠标点击**：在所选列放置棋子
- **R键**：重新开始游戏

#### 游戏规则
- 玩家轮流在7列中选择一列放置棋子
- 棋子会落在所选列的最底部
- 先连成4子的玩家获胜
- 棋盘填满仍未分胜负，则为平局

### 7. 记忆卡牌游戏（开发中）
#### 游戏目标（规划）
在限定时间内，找出所有两两配对的卡牌，考验短期记忆力和观察力。

#### 开发计划
- 第一阶段：实现基础卡牌翻牌与配对逻辑
- 第二阶段：添加计时与分数系统
- 第三阶段：支持自定义卡牌主题和难度

## 常见问题

### 1. 游戏无法运行，提示找不到主类

**解决方法**：确保已正确编译项目，并且类路径设置正确。使用命令行运行时，需要确保已创建bin目录并将编译后的class文件放入其中。

### 2. 游戏界面显示异常

**解决方法**：确保使用的是Java 8或更高版本，因为项目使用了Java Swing的一些特性。

### 3. 游戏运行缓慢

**解决方法**：关闭其他占用系统资源的程序，或尝试降低游戏窗口大小（如果支持）。

### 4. 如何添加新游戏

**解决方法**：在`src/games`目录下创建新的子目录，然后实现游戏的主类和相关组件。建议遵循README.md中的开发说明。
